# 智能测评系统 - 技术实施方案

## 项目概述

面向素质教育机构的智能入学测评系统，支持1-9年级学生的AIGC素养课和编程课入学测评，智能生成试卷和报告，实现招生分班自动化。

---

## 数据库设计

### 核心数据表

#### 1. 用户表 (users)
```sql
- id, username, password, phone, role(student/teacher/admin)
- name, gender, birthday, school, grade, avatar
- created_at, updated_at, status
```

#### 2. 课程表 (courses)
```sql
- id, name, code, category(aigc/programming), description
- grade_range, status, created_at
```

#### 3. 题库表 (questions)
```sql
- id, course_id, grade, type(single/multi/judge/blank)
- content, options(JSON), answer, analysis
- difficulty(1-5), dimensions(逻辑/创意/基础等)
- source, status(pending/approved/rejected)
- created_by, created_at, reviewed_at, reviewed_by
```

#### 4. 测评卷配置表 (exam_papers)
```sql
- id, name, course_id, grade, total_questions, time_limit
- dimensions(JSON: 各维度权重), status, created_at
```

#### 5. 测评记录表 (exam_records)
```sql
- id, student_id, exam_paper_id, course_id
- answers(JSON), score, status
- started_at, submitted_at
- ai_analysis, ability_level
```

#### 6. 能力等级配置表 (ability_levels)
```sql
- id, course_id, grade, level(A/B/C/D)
- min_score, max_score, description, suggestions(JSON)
```

#### 7. 录取通知表 (admission_notices)
```sql
- id, student_id, course_id, exam_record_id
- template_id, content(JSON), status
- sent_at, created_by
```

#### 8. 通知模板表 (notice_templates)
```sql
- id, name, type(admission/notice), content(JSON)
- variables(JSON), created_by, updated_at
```

#### 9. 班级表 (classes)
```sql
- id, name, course_id, teacher_id
- grade, student_count, created_at
```

## 功能模块详细设计

### 1. 学生端功能

#### 1.1 测评流程
```
信息填写 → 课程选择 → 智能组卷 → 在线答题 → 提交测评 → 查看报告 → 接收通知
```

**信息收集表单**：
- 姓名、性别、出生年月
- 在读学校、在读年级
- 数学成绩（优/良/中/差）
- AI/编程基础（零基础/初级/中级/高级）
- 获奖情况（无/校级/区级/市级/省级/国家级）

#### 1.2 智能组卷引擎
```javascript
组卷算法：
1. 根据学生年级 + 课程类别 + 能力基础筛选题目池
2. 按考核维度分配题目数量（基础40%/逻辑30%/创意30%）
3. 随机抽取确保多样性
4. AI辅助调整难度
5. 生成唯一试卷
```

#### 1.3 个人报告
- 个人信息汇总
- 测评数据雷达图
- 各维度得分详情
- 能力等级评估
- 学习建议（个性化）
- 推荐课程

### 2. 教师端功能

#### 2.1 学生管理
- 批量导入学生信息
- 查看学生测评进度
- 导出学生数据

#### 2.2 测评管理
- 查看班级测评汇总
- 查看单个学生报告
- 下发录取通知

#### 2.3 通知书管理
- 选择通知模板
- 批量发送
- 记录发送状态

### 3. 管理端功能

#### 3.1 用户管理
- 用户CRUD
- 角色权限配置
- 批量操作

#### 3.2 测评配置
- 测评卷管理（创建/编辑/启用）
- 题目数量配置（每卷15-30题可调）
- 题型配置
- 考核维度权重
- 能力等级阈值

#### 3.3 题库管理
- 题目CRUD
- AI辅助出题（生成题目建议）
- 人工审核流程
- 批量导入/导出
- 题目分类标签

#### 3.4 通知模板配置
- 模板变量设置
- 样式编辑
- 预览效果

## 题库设计（200道题目）

### 按课程分类

#### AIGC素养课（100道）
| 年级 | 题型分布 | 题目数量 |
|------|----------|----------|
| 1-3年级 | 单选、判断 | 25 |
| 4-6年级 | 单选、多选、判断 | 40 |
| 7-9年级 | 单选、多选、填空、判断 | 35 |

#### 编程课（100道）
| 年级 | 题型分布 | 题目数量 |
|------|----------|----------|
| 1-3年级(Scratch) | 单选、判断、简答 | 25 |
| 4-6年级(Python) | 单选、多选、填空 | 40 |
| 7-9年级(C++) | 单选、多选、填空、编程 | 35 |

### 考核维度
- **基础认知**：基本概念、术语理解
- **逻辑思维**：问题分析、推理判断
- **创意应用**：创新想法、实践能力
- **综合素养**：跨学科、知识迁移

- 支持100+并发

- 题库≥200道题目
- 覆盖1-9年级

## 技术亮点

1. **智能组卷算法**：基于维度和难度动态抽题
2. **AI辅助出题**：OpenAI API + 本地规则双重保障
3. **可视化报告**：ECharts图表 + PDF导出
4. **响应式设计**：适配PC/平板/手机
5. **降级机制**：AI服务不可用时自动切换本地逻辑
